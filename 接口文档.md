# Campus Lost-Found 校园失物招领平台 接口文档

## 基本信息

| 项目 | 说明 |
|------|------|
| 项目名称 | Campus Lost-Found 校园失物招领平台 |
| 后端技术 | Spring Boot + MyBatis-Plus + MySQL |
| 前端技术 | Vue.js |
| 后端端口 | 8080 |
| 基础URL | `http://localhost:8080/api` |

---

## 认证说明

### Token 认证

除以下**公开接口**外，其他接口均需要在请求头中携带 Token：

```
Authorization: Bearer <token>
```

### 公开接口（无需认证）

| 接口 | 说明 |
|------|------|
| `POST /api/users/login` | 用户登录 |
| `POST /api/users/register` | 用户注册 |
| `POST /api/users/send-code` | 发送验证码 |
| `GET /api/users/list` | 获取用户列表（聊天联系人） |
| `GET /api/posts` | 获取所有帖子 |
| `GET /api/posts/type/{type}` | 按类型获取帖子 |
| `GET /api/posts/search` | 综合检索帖子 |
| `GET /api/comments/post/{postId}` | 获取帖子评论 |
| `GET /api/buildings` | 获取建筑字典 |
| `POST /api/chat/send` (WebSocket) | WebSocket 聊天接口 |
| `/uploads/**` | 静态资源访问 |

### 管理员接口

管理员接口需要 `Authorization` 头中携带 **ADMIN** 角色的 Token：

```
Authorization: Bearer <admin_token>
```

## 2. 帖子模块 `/api/posts`

### 2.1 发布帖子

**请求**

```
POST /api/posts
Authorization: Bearer <token>
Content-Type: application/json
```

**请求体**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| type | String | 是 | 类型：LOST（寻物）/ FOUND（招领） |
| title | String | 是 | 标题 |
| description | String | 否 | 详细描述 |
| contact | String | 否 | 联系方式 |
| imageUrl | String | 否 | 图片 URL |
| itemStatus | String | 否 | 物品状态：PENDING / RESOLVED |
| incidentStartDate | String | 否 | 起始日期（格式：YYYY-MM-DD） |
| incidentEndDate | String | 否 | 结束日期（格式：YYYY-MM-DD） |
| incidentTimeDesc | String | 否 | 模糊时间描述 |
| buildingId | Long | 否 | 建筑物 ID |
| locationDesc | String | 否 | 具体位置描述 |
| latitude | Double | 否 | 纬度 |
| longitude | Double | 否 | 经度 |
| verifyQuestion | String | 否 | 验证问题 |
| verifyAnswer | String | 否 | 验证答案 |

**响应示例**

```json
{
  "id": 1,
  "type": "LOST",
  "title": "丢失黑色双肩包",
  "description": "昨天下午在图书馆丢失...",
  "contact": "13800138000",
  "imageUrl": "http://localhost:8080/uploads/123.jpg",
  "itemStatus": "PENDING",
  "incidentStartDate": "2024-03-15",
  "incidentEndDate": "2024-03-15",
  "incidentTimeDesc": "下午二节课后",
  "buildingId": 1,
  "locationDesc": "图书馆二楼靠窗座位",
  "latitude": 30.123,
  "longitude": 114.456,
  "createTime": "2024-03-15T16:30:00",
  "publisherId": "1"
}
```

**路径参数**
